# 公众号文章内容审查清单

## 一、文字审查

### 1.1 错别字检查
- [ ] 同音字混淆（如：的/地/得、在/再、已/以）
- [ ] 形近字混淆（如：末/未、己/已/巳）
- [ ] 专业术语拼写
- [ ] 人名/品牌名/公司名准确性
- [ ] 英文单词拼写

### 1.2 标点符号规范
- [ ] 中英文标点混用
- [ ] 句号/逗号使用恰当
- [ ] 引号配对完整
- [ ] 括号配对完整
- [ ] 省略号使用规范（中文用……，不用...）

### 1.3 语句通顺性
- [ ] 主谓宾搭配正确
- [ ] 没有语病（成分残缺、搭配不当、语序不当等）
- [ ] 指代明确
- [ ] 逻辑连贯

### 1.4 前后一致性
- [ ] 术语使用统一（同一概念不要用不同称呼）
- [ ] 人称使用统一
- [ ] 时态使用统一
- [ ] 格式风格统一（如编号方式）

## 二、事实核查

### 2.1 数据准确性
- [ ] 数字是否有明确来源
- [ ] 统计数据是否来自权威机构
- [ ] 计算结果是否正确
- [ ] 单位是否准确（亿/万、美元/人民币）

### 2.2 引用来源可靠性
- [ ] 引用的观点是否标注出处
- [ ] 引用来源是否为权威机构/人物
- [ ] 引用内容是否准确（未断章取义）
- [ ] 翻译内容是否准确

### 2.3 时效性检查
- [ ] 信息是否过时
- [ ] 产品功能是否已更新
- [ ] 政策/规则是否已变化
- [ ] 链接是否仍然有效

## 三、法律风险审查

### 3.1 版权问题
- [ ] 图片是否有版权（优先使用自制图/授权图/无版权图）
- [ ] 引用内容是否超过合理使用范围
- [ ] 是否标注了必要的来源
- [ ] 是否涉及未授权使用他人作品

### 3.2 商标/品牌名称使用
- [ ] 品牌名称拼写正确
- [ ] 使用方式是否合规（如 ®、™ 标识）
- [ ] 是否有贬低或诋毁其他品牌的内容
- [ ] 是否存在虚假宣传或比较广告问题

### 3.3 个人隐私保护
- [ ] 是否涉及未经授权的个人信息
- [ ] 截图中是否隐藏了敏感信息（ID、联系方式等）
- [ ] 案例中的人物是否需要匿名处理
- [ ] 是否涉及未成年人信息

### 3.4 敏感话题规避
- [ ] 是否涉及政治敏感话题
- [ ] 是否涉及宗教敏感话题
- [ ] 是否涉及民族/地域歧视
- [ ] 是否涉及性别歧视
- [ ] 是否涉及未经证实的传闘

## 四、负面影响评估

### 4.1 是否可能引起误解
- [ ] 标题是否有标题党嫌疑
- [ ] 内容是否容易被断章取义
- [ ] 观点是否过于绝对化
- [ ] 是否有模糊或歧义的表达

### 4.2 是否有不当类比或措辞
- [ ] 类比是否恰当（避免不当比喻）
- [ ] 是否使用了可能冒犯的词汇
- [ ] 是否有居高临下的语气
- [ ] 是否有不尊重读者的表达

### 4.3 是否涉及敏感群体
- [ ] 是否可能伤害特定群体
- [ ] 是否有刻板印象表达
- [ ] 是否公平对待不同观点
- [ ] 是否尊重多元性

### 4.4 是否有过度承诺
- [ ] 效果描述是否过于夸大
- [ ] 是否有"绝对""一定"等绝对化用语
- [ ] 收益描述是否客观
- [ ] 风险提示是否充分

## 五、合规性检查

### 5.1 广告法合规
- [ ] 是否使用了禁用词（最、第一、国家级、顶级等绝对化用语）
- [ ] 是否有虚假宣传
- [ ] 是否明确标注了广告/推广内容
- [ ] 是否有误导消费者的内容

### 5.2 互联网内容规范
- [ ] 是否符合《网络信息内容生态治理规定》
- [ ] 是否有低俗/暴力/血腥内容
- [ ] 是否有诱导分享内容
- [ ] 是否有恶意营销内容

### 5.3 行业特定规范
- [ ] 金融类内容是否合规
- [ ] 医疗健康类内容是否有风险提示
- [ ] 教育类内容是否客观
- [ ] 技术类内容是否准确

## 七、常见问题速查

### 高频错别字
| 错误 | 正确 | 说明 |
|------|------|------|
| 的/地/得 | 根据语法规则 | 的+名词，地+动词，得+形容词 |
| 在/再 | 根据语义 | 在=存在/位置，再=又/继续 |
| 已/以 | 根据语义 | 已=已经，以=用/凭借 |
| 做/作 | 根据搭配 | 做+具体动作，作+抽象名词 |

### 广告法禁用词清单
- 绝对化用语：最、第一、唯一、首个、顶级、极品、最佳
- 虚假表述：国家级、世界级、全网销量第一
- 诱导性用语：史无前例、万人疯抢、错过后悔

### 敏感话题红线
- 国家政治、领导人
- 宗教信仰
- 民族/地域歧视
- 未经证实的负面新闻
- 暴力、血腥、色情内容
