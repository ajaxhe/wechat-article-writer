# 公众号配图生成提示词模板

## 一、封面图生成

### 1.1 标准封面图规格

**微信公众号封面图尺寸：**
- 头条封面：900 × 383 像素（2.35:1）或 1080 × 459 像素
- 次条封面：200 × 200 像素（正方形）
- 分享封面：500 × 400 像素

### 1.2 封面图风格指南

**推荐风格：**
- 简约科技风：适合AI、技术类文章
- 温暖生活风：适合育儿、生活类文章
- 商务专业风：适合行业分析、职场类文章
- 手绘插画风：适合观点表达、故事类文章

**配色建议：**
- 科技蓝：#0066FF、#00AAFF
- 暖橙色：#FF6B35、#FFB347
- 深邃紫：#6B5B95、#9B59B6
- 清新绿：#2ECC71、#27AE60
- 专业灰：#34495E、#7F8C8D

## 二、内容配图生成

### 2.1 概念解释图

用于解释抽象技术概念或原理

**提示词模板：**
```
Create an explanatory diagram for the concept of [概念名称]:
- Purpose: Help readers understand [具体要解释的内容]
- Style: Clean infographic style, educational
- Visual metaphor: [类比物，如：将Agent比作助手]
- Elements: 
  - Core concept in the center
  - Related components around it
  - Connection lines showing relationships
- Color coding: Use colors to differentiate categories
- Annotations: Space for labels (will add text separately)
- Size: 800x600 pixels
- Background: White or light gray
```

### 2.2 流程图/步骤图

展示工作流程或操作步骤

**提示词模板：**
```
Create a process flow diagram showing [流程名称]:
- Number of steps: [步骤数量]
- Flow direction: [从左到右/从上到下]
- Style: Modern, clean, with [图标风格] icons
- Each step should have:
  - Icon representation
  - Clear visual boundary
  - Connection to next step
- Color scheme: Progressive colors showing advancement
- Size: 900x500 pixels
- Layout: [线性/循环/分支]
```

### 2.3 对比图

展示两种方案/方法的差异

**提示词模板：**
```
Create a comparison visual showing [对比主题]:
- Left side: [方案A名称] - representing [特点]
- Right side: [方案B名称] - representing [特点]
- Style: Split-screen design
- Visual contrast:
  - [方案A用什么视觉元素表示]
  - [方案B用什么视觉元素表示]
- Common elements in the middle showing transition
- Color scheme: [方案A颜色] vs [方案B颜色]
- Size: 800x500 pixels
- Clear visual hierarchy showing improvement/difference
```

### 2.4 数据可视化图

展示数据趋势或统计信息

**提示词模板：**
```
Create a data visualization showing [数据主题]:
```

## 四、配图使用指南

### 4.1 配图原则

1. **相关性**：图片必须与内容直接相关
2. **简洁性**：避免过于复杂的视觉元素
3. **一致性**：同一篇文章的配图风格要统一
4. **可读性**：图片中的信息要清晰可辨
5. **原创性**：优先使用AI生成的原创图片

### 4.2 配图位置建议

- **开头**：封面图+引言配图
- **每个主要章节**：1张说明性配图
- **数据展示处**：数据可视化图
- **步骤说明处**：流程图或分步骤图
- **结尾**：可选的总结图或行动引导图

### 4.3 图片数量建议

| 文章长度 | 建议配图数量 |
|---------|-------------|
| 1000字以内 | 1-2张 |
| 1000-2000字 | 2-4张 |
| 2000-3000字 | 3-5张 |
| 3000字以上 | 4-6张 |

### 4.4 图片优化建议

1. **压缩**：确保图片大小适合网络加载
2. **尺寸**：宽度建议不超过1080像素
3. **格式**：JPG用于照片，PNG用于图标/透明图
4. **Alt文本**：为图片添加描述文字（可选）

## 五、风格参考

### 5.1 科技类文章配图风格

**特点：**
- 蓝色/紫色为主色调
- 几何图形、网格线
- 渐变和光效
- 简洁的图标和线条

**关键词：**
futuristic, minimal, clean lines, gradient, geometric, tech, digital

### 5.2 生活类文章配图风格

**特点：**
- 暖色调为主
- 手绘/插画风格
- 生活场景元素
- 温馨的光线

**关键词：**
warm, cozy, lifestyle, illustration, soft colors, home, family

### 5.3 商业类文章配图风格

**特点：**
- 专业的配色方案
- 数据可视化元素
- 商务场景
- 干净的背景

**关键词：**
professional, corporate, business, data, charts, modern office
